## Installation

Using **npm**:

```
$ npm install --save gh-explore
```

If you don't have or don't want to use **npm**:

```
$ cd ~/.node_modules
$ git clone git://github.com/gmontalvoriv/gh-explore.git
```

## Test

Using **npm**:

```
$ npm test
```

Using **make**:

```
$ make test
```
